ReFT

块级残差流干预（Wu et al., NeurIPS 2024）。在选定的每个 DiT 块上添加一个 ReFTModule，将 R^T · (ΔW·h + b) · scale 添加到块输出上 — 这不是 Linear 的权重差分，而是额外的旁路通道。可与所有 LoRA 变体组合使用，保存在同一个 .safetensors 文件中。

通过 reft_layers 选择块（例如 last_8、stride_2 或逗号分隔的索引）。reft_dim 设置干预 rank，reft_alpha 通过 alpha / dim 设置有效缩放。

tlora_ortho_reft 是将 ReFT 与 T-LoRA + OrthoLoRA 打包到一次训练中的变体。LoRA 部分可烘焙，但 ReFT 块钩子无法烘焙，合并后的 DiT 中 ReFT 贡献会消失（Merge 工具会发出警告并要求 --allow-partial）。
