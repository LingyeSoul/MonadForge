T-LoRA

时间步依赖的秩掩码。有效 LoRA rank 随去噪时间步按指数调度变化：
• 高噪声（前期步） → 完整 rank（最大表达力）
• 低噪声（后期步） → 缩减 rank（至 min_rank）
在结构确定的阶段集中表达力。掩码仅在训练时应用，推理时所有 t 均使用完整 rank，因此保存的 .safetensors 是可烘焙的标准 LoRA。参见 docs/methods/timestep_mask.md。

开关：use_timestep_mask = true。

变体
• tlora — T-LoRA + OrthoLoRA 组合（use_ortho = true + use_timestep_mask = true）。默认可烘焙组合 — 同时应用 σ 级秩调度和正交正则化。可完全烘焙。
• tlora-8gb — 相同架构，低显存配置（梯度检查点 + 卸载）。如果默认变体出现 OOM，请选择此项。
